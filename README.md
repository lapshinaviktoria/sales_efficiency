# Проект по анализу динамики продаж маркетплейса

### Описание проекта
Цель проекта: создать дашборд для возможности быстро оценивать и контролировать динамику продаж, активность пользователей и качество товаров для принятия оперативных бизнес-решений.	

В репозитории содержится:
- техническое задание
- докуметнация для обучения пользователей
- демонстрация дашборда в pdf-формате
